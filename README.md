mecab-swift
===========

MeCab wrapper for Swift.


```swift 
import MeCab

let m = Mecab()

let nodes = try! m.tokenize("太郎は次郎が持っている本を花子に渡した。")

for n in nodes.filter({ !$0.isBosEos }) {
    print(n.surface, n.features)
}      
```
